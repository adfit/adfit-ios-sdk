AdFit iOS SDK

이 가이드는 iOS Application 에 모바일 광고를 노출하기 위한 광고 데이터요청과 처리 방법을 설명합니다.
사이트/앱 운영정책에 어긋나는 경우 적립금 지급이 거절 될 수 있으니 유의하시기 바랍니다.

  • 문의 고객센터 https://cs.kakao.com/helps?service=160&locale=ko
  • 사이트/앱 운영 정책 https://adfit.kakao.com/web/html/use_kakao.html

지원 환경

AdFitSDK는 Swift 기반으로 개발되었습니다.
Swift 기반의 프로젝트에서 AdFitSDK를 사용하시려면, 반드시 최신 버전의 Xcode를 사용해주세요.

  • 최신 버전의 Xcode (Xcode 13. / Swift 5.0)
  • iOS Deployment Target: iOS 12.0 이상
